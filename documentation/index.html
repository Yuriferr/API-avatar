API de Avatar: A Lenda de Aang

API Avatar é uma interface de programação de aplicações que permite aos usuários acessar informações sobre o universo da série animada Avatar: A Lenda de Aang, desenho criado por Michael Dante DiMartino e Bryan Konietzko. A API foi desenvolvida em Python e utiliza o formato JSON para troca de dados.
Para usar a API Avatar, os usuários devem enviar requisições HTTP para a URL base da API, seguida da rota desejada e dos parâmetros opcionais.

versão 1.0

Rota base

A rota base é /. Ela retorna um objeto JSON com informações gerais sobre as quatro nações e os personagens principais de cada nação. O formato do objeto é o seguinte:

http://avatarapi.pythonanywhere.com

Exemplo de retorno dos dados da rota base: 

          
            [
              {
                "id": 1,
                "nome": "Tribo da água",
                "imagem": "exemple/image/water.png",
                "personagens": [
                  {
                    "id": 1,
                    "nome": "Katara",
                    "idade": 14,
                    "dobra": ["Água", "Sangue", "Gelo"],
                    "nação": "Tribo da água",
                    "imagem": "exemple/image/katara.png"
                  },
                  {
                    "id": 2,
                    "nome": "Sokka",
                    "idade": 15,
                    "dobra": ["Nenhuma"],
                    "nação": "Tribo da água",
                    "imagem": "exemple/image/sokka.png"
                  }
                }
              ]

Rota dobras

A rota dobras é /dobras. Ela retorna um objeto JSON com informações sobre os quatro elementos que podem ser dobrados pelos personagens da série: ar, água, terra e fogo. O formato do objeto é o seguinte:

http://avatarapi.pythonanywhere.com/dobras

Exemplo de retorno dos dados da rota dobras: 

          
            [
              {
                  "id": "1",
                  "name": "Água",
                  "description": "Dobra básica do elemento água"
              },
              {
                  "id": "2",
                  "name": "Terra",
                  "description": "Dobra básica do elemento terra"
              },
              {
                  "id": "3",
                  "name": "Fogo",
                  "description": "Dobra básica do elemento fogo"
              },
              {
                  "id": "4",
                  "name": "Ar",
                  "description": "Dobra básica do elemento ar"
              }
            ]

Rota nações

A rota nações é /avatar/nacoes. Ela retorna um objeto JSON com informações sobre as quatro nações e as províncias de cada nação. O formato do objeto é o seguinte:

http://avatarapi.pythonanywhere.com/nacoes

Exemplo de retorno dos dados da rota nações:
